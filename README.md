## 개발 목적

- React와 fireBase를 활용하여 실시간 통신이 되는 커뮤니티 사이트를 만든다.


## 구현 기능

- FireBase의 Authentication를 활용한 로그인 기능
- FireBase의 FireBase 활용한 CRUD기능
- 등등 업데이트예정
